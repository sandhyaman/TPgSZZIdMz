# Python旅游景点数据分析与推荐系统

## 核心功能
- 登录注册
- 旅游景点推荐
- 新闻资讯
- 系统简介
- 今日头条
- 旅游景点管理
- 个人中心
- 用户管理
- 管理员管理
- 系统管理

## 适用场景
- 旅游爱好者
- 旅游机构
- 旅游规划师
- 地方旅游管理部门

## 技术特点
- 采用Django和Vue.js构建，前端后端分离
- 基于MySQL数据库进行数据存储
- 提供远程调试支持
